# Kash OS Demo

A static, modern web dashboard and login/landing page demo for "Kash OS".

- **index.html** - Minimal login/landing page (Astro + Supabase starter style)
- **dashboard.html** - Dashboard with sidebar, widgets, profile, and app list

Preview and customize these for your own use. No backend required—static HTML/CSS only.

```
Open index.html or dashboard.html in your web browser.
```
